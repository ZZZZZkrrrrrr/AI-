# Release Evidence Folder

Use this folder only for sanitized launch and store-submission evidence.

Allowed examples:

- Redacted command result summaries.
- Redacted public-domain smoke notes.
- Redacted screenshots that do not show account, billing, private user, local path, credential, or signing material.
- Short checklist notes that point to external private evidence without exposing secrets.

Do not store:

- Passwords, API keys, cookies, keystores, p12 files, provisioning profiles, or signing passwords.
- Private customer uploads or generated customer outputs.
- Unredacted App Store Connect, Google Play Console, hosting dashboard, billing, logs, or support ticket screenshots.
- Identity documents, business license scans, or private account recovery information.
