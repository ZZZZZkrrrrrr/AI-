安装到手机桌面

把网页添加到手机桌面后，可以像打开 App 一样进入工作台。用户不需要理解 PWA，只要按自己的手机系统完成下面几步。

推荐域名：https://www.zkraiflow.top/ 更新日期：2026-06-10
打开 App 帮助与支持 隐私政策 删除账号与数据

先确认这 3 件事

  1. 用手机浏览器打开 https://www.zkraiflow.top/。
  2. 确认页面可以正常进入“生产总览”或登录页。
  3. 如果已经打开旧页面，先刷新一次，再按下面步骤安装。

iPhone / iPad

  1. 使用 Safari 打开 www.zkraiflow.top。
  2. 点击 Safari 底部或顶部的分享按钮。
  3. 在菜单里选择“添加到主屏幕”。
  4. 名称保持“AI 视频工作台”，点击“添加”。
  5. 回到桌面，点击新图标打开。

Android 手机

  1. 使用 Chrome 打开 www.zkraiflow.top。
  2. 如果页面底部或顶部出现“安装”提示，直接点击安装。
  3. 如果没有提示，点击 Chrome 右上角菜单。
  4. 选择“安装应用”或“添加到主屏幕”。不同手机文字可能略有差异。
  5. 安装完成后，从桌面或应用列表打开。

安装后怎么判断成功

  • 从桌面图标打开后，底部有“首页、创建、素材、我的”导航。
  • 右上角通知按钮可以看到任务提醒。
  • “我的”页面可以看到 App 准备中心、隐私政策、帮助与支持、账号数据入口。
  • 断网时会显示离线提示页，恢复网络后刷新即可继续使用。
