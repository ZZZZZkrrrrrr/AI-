# 全品类商品提示词拆解：通用复用模型

目标：把“男装/女装服装带货提示词包”的写法，抽象成所有商品都可以复用的提示词规范包。

适用范围：

- 服装、鞋包、饰品、珠宝
- 美妆、护肤、香水、个护
- 食品、饮品、保健类普通消费品
- 家居、家电、厨具、收纳、清洁用品
- 数码、配件、办公用品
- 母婴、宠物、户外、运动、礼品
- 门店、课程、服务型商品

## 一句话原则

不是直接介绍商品，而是让用户看到：

> 这个商品进入我的生活后，会解决什么问题、带来什么状态、提升什么体验、为什么值得买。

所以所有商品提示词都可以用同一个底层逻辑：

```text
商品本体
-> 商品属性锁定
-> 目标人群
-> 购买动机
-> 使用场景
-> 使用动作 / 体验证据
-> 视觉表达
-> 时间轴
-> 最终视频提示词
```

## 通用底层公式

```text
商品提示词规范包 =
版本边界
+ 全局规则
+ 商品读图/读信息建档
+ 商品类别判断
+ 专家角色激活
+ 用户购买心理判断
+ 卖点转场景
+ 场景/道具/动作匹配
+ 摄影/镜头/声音设计
+ 目标时长时间轴
+ 最终提示词模板
+ 负面提示词
+ 样片示例
```

这套结构的关键不是“商品是什么”，而是“商品如何被用户感知为有用、有欲望、有必要”。

## 从服装版迁移到全商品版

| 服装版模块 | 全商品通用模块 |
|---|---|
| 服装类型、颜色、版型、材质 | 商品类型、外观、规格、材质、结构、包装 |
| 完整穿搭/完整造型 | 完整使用方案/完整消费场景 |
| 男性气质/女性气质 | 用户状态/生活改善/身份感/情绪价值 |
| 车辆/咖啡店/约会/通勤 | 商品最适合出现的真实使用场景 |
| 鞋包首饰妆发 | 配套物品、使用道具、环境物件 |
| 显高显瘦/漂亮/帅气 | 商品带来的核心结果：方便、好看、好吃、舒适、省时、专业、体面、安心 |
| 服装锁定规则 | 商品锁定规则：外观、颜色、形状、包装、结构、品牌信息、功能边界 |
| 15 秒穿搭分镜 | 15 秒商品体验分镜 |

## 第三步：购买心理通用判断

所有商品都要先判断用户为什么会买。常见购买动机如下：

| 购买动机 | 适合商品 | 视频表达 |
|---|---|---|
| 变好看 | 服装、美妆、饰品、家居装饰 | 人物状态、空间氛围、细节质感 |
| 更方便 | 小家电、工具、数码、收纳 | 使用步骤减少、动作顺滑、省力 |
| 更舒服 | 家居、个护、服饰、母婴 | 触感、放松状态、安静场景 |
| 更专业 | 办公、数码、课程、工具 | 工作流、效率、清晰结果 |
| 更体面 | 礼品、珠宝、服装、商务用品 | 送礼、见面、通勤、宴请场景 |
| 更安心 | 母婴、宠物、个护、食品 | 干净、安全感、清楚边界 |
| 更好吃/好喝 | 食品、饮品 | 质地、热气、咀嚼/倒饮、分享 |
| 更省时间 | 家电、清洁、工具、服务 | 对比动作、快速完成、结果镜头 |
| 更有情绪价值 | 香水、礼物、文创、生活方式商品 | 氛围、仪式感、音乐、光影 |

不要只写“高级”“好用”“好看”，要写成：

```text
目标用户：
原本痛点：
商品介入：
使用后的状态：
购买理由：
```

## 第五步：卖点转画面的方法

商品卖点不能停留在抽象词，要转成可拍摄证据。

| 抽象卖点 | 可拍画面 |
|---|---|
| 质感好 | 微距特写、侧光高光、手部触摸、材质纹理 |
| 方便 | 一只手完成、步骤减少、随手拿取、快速收纳 |
| 显效果 | 使用前状态 -> 使用中动作 -> 使用后结果 |
| 好吃 | 掰开、倒出、热气、咬下、质地拉丝/酥脆/绵密 |
| 好看 | 空间陈列、人物使用、色彩搭配、细节定格 |
| 耐用 | 结构细节、连接处、材质厚度、重复使用动作 |
| 适合送礼 | 包装打开、礼盒摆放、递给对方、收礼表情 |
| 省空间 | 使用前杂乱、放入商品、空间变整齐 |
| 专业 | 工作台、工具配合、流程清楚、结果干净 |

| 商品类型 | 感官主线 | 镜头重点 |
|---|---|---|
| 服装鞋包 | 身体比例、材质、搭配、动作 | 全身、细节、走动、场景定格 |
| 美妆护肤 | 质地、肤感、光泽、妆效 | 挤出、涂抹、上脸、局部变化 |
| 食品饮品 | 香气、口感、温度、质地 | 倒出、掰开、咬下、热气、杯壁 |
| 家居家电 | 空间秩序、使用效率、结果 | 前后对比、手部操作、空间全景 |
| 数码产品 | 操作流畅、屏幕内容、便携 | 开机、连接、点击、手持使用 |
| 珠宝饰品 | 光泽、佩戴、礼物感 | 微距、佩戴动作、光线反射 |
| 服务课程 | 流程、信任、结果状态 | 现场、步骤、用户状态、成果 |

## 第八步：9:16 商品视频摄影规则

通用硬约束：

- 所有镜头优先按 9:16 竖屏设计。
- 商品必须在手机屏幕中部安全区清晰出现。
- 开场 2 秒必须让用户知道“这是什么商品”或“它解决什么问题”。
- 不能让人物、背景、道具抢走商品主体。
- 产品细节至少出现一次清晰特写。
- 使用结果至少出现一次清晰展示。
- 不要先设计横屏画面再裁成竖屏。

通用景别：

| 景别 | 用途 |
|---|---|
| 极近景 | 材质、质地、包装、logo、结构细节 |
| 近景 | 手部操作、涂抹、打开、倒出、拿取 |
| 中景 | 人物与商品关系、使用动作 |
| 中远景 | 商品进入生活场景 |
| 全景 | 最终结果、空间氛围、使用后状态 |

通用运镜：

| 运镜 | 用途 |
|---|---|
| 慢推 | 强化商品细节和欲望感 |
| 后拉 | 从商品细节扩展到完整场景 |
| 跟拍 | 展示使用过程 |
| 俯拍 | 展示桌面、食物、收纳、开箱 |
| 低角度 | 强化体积、质感、人物状态 |
| 静态定格 | 建立高级、干净、可复制的记忆点 |

## 第九步：15 秒通用时间轴

```text
0-2 秒：开场钩子
用商品细节、使用痛点、打开动作、效果结果先抓住注意力。

2-4 秒：商品识别
清楚展示商品外观、包装、颜色、结构或核心部件。

4-6 秒：核心卖点证据
把一个最重要卖点转成可见动作或可见结果。

6-9 秒：使用过程
展示手部操作、人物使用、场景嵌入或前后变化。

9-12 秒：使用后状态
展示用户变好看、更方便、更舒服、更整洁、更专业、更有食欲等结果。

12-15 秒：记忆点定格
商品与场景同框，形成购买理由，不出现硬促销贴纸。
```

不同商品可以替换开场钩子：

| 商品类型 | 0-2 秒钩子 |
|---|---|
| 服装 | 衣料细节、穿上瞬间、走动入镜 |
| 美妆 | 质地挤出、刷头接触、上脸前后 |
| 食品 | 掰开、倒出、热气、第一口 |
| 家电 | 一键启动、问题状态出现、快速完成 |
| 家居 | 杂乱空间 -> 商品入场 |
| 数码 | 开机亮屏、磁吸连接、快捷操作 |
| 珠宝 | 打开礼盒、光泽闪动、佩戴瞬间 |
| 服务 | 用户痛点画面、现场接待、结果预览 |

## 第十步：最终提示词通用模板

```text
生成一条 [时长]、9:16 竖屏、无口播、无电商贴纸的商品种草短视频。

商品类型：[填写商品品类]
商品识别：[颜色、外观、材质、包装、结构、规格]
商品锁定规则：[必须保持不变的外观、结构、包装、颜色、品牌信息]

目标用户：[谁会买]
购买动机：[为什么买]
核心场景：[商品出现在哪里]
核心体验：[使用后带来的状态或结果]

画面风格：[生活方式 / 电影感 / 干净电商 / 高级质感 / 真实使用 / 温暖家庭 / 专业效率]
人物与动作：[谁在使用，怎么使用]
道具与环境：[配套物品、空间、光线]

镜头结构：
0-2 秒：[开场钩子]
2-4 秒：[商品识别]
4-6 秒：[卖点证据]
6-9 秒：[使用过程]
9-12 秒：[使用后状态]
12-15 秒：[商品与场景定格]

摄影要求：
9:16 竖屏构图，商品始终在画面安全区内清晰可见。
至少一次商品细节特写，至少一次使用过程，至少一次使用结果。
使用适合商品质感的光线、景别和运镜。

声音：
只保留环境音、商品使用声、轻微动作声和适合风格的 BGM。

禁止：
口播、主持人讲解、价格贴纸、购物车图标、促销大字、横屏构图、商品变形、颜色改变、包装改变、功能夸大、虚假效果、廉价滤镜、画面脏乱、无关人物抢主体。
```

## 负面提示词通用库

| 类别 | 负面提示词 |
|---|---|
| 内容形式 | 无口播、无主持人、无直播间、无采访感、无强促销字幕 |
| 电商元素 | 无价格贴纸、无购物车、无限时促销、无夸张 CTA |
| 商品还原 | 不改变颜色、形状、包装、结构、材质、品牌信息、规格比例 |
| 功能边界 | 不夸大效果、不制造虚假前后对比、不暗示未经证明的功效 |
| 画面质量 | 不要廉价滤镜、不要过曝、不要过度磨皮、不要脏乱背景 |
| 人物表现 | 不要夸张表演、不要低质短剧感、不要油腻摆拍、不要手部畸形 |
| 构图 | 不要横屏 16:9、不要宽银幕黑边、不要商品被裁切、不要主体太小 |

## 全商品写作检查清单

写一个商品提示词包前，必须先回答：

1. 这个提示词包只服务哪一种视频风格？
2. 这个商品最容易被 AI 生成错的地方是什么？
3. 用户为什么会买它？
4. 它应该出现在什么真实生活场景？
5. 它的卖点能不能被拍成动作或结果？
6. 哪些功能不能夸大？
7. 开场 2 秒用什么抓人？
8. 15 秒内至少哪三个镜头必须出现？
9. 最终提示词里是否写入商品锁定规则？
10. 负面提示词是否覆盖跑偏风险？

## 最终可复用骨架

以后任何商品都按这条主线拆：

```text
边界
-> 商品建档
-> 用户心理
-> 专家角色
-> 卖点可视化
-> 场景匹配
-> 动作/道具设计
-> 摄影语言
-> 时间轴
-> 商品锁定
-> 负面提示词
-> 样片校准
```

服装只是其中一个类目。

真正可复用的是这套“商品进入生活场景后创造购买理由”的结构。
